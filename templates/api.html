Currently, version 1.0 of the API is stable, but we are working on a higher quality version 2.0, made with lessons learned. Have a look at the schemas and do not forget to append a format paramater to the url you are requesting, e.g. "?format=json". To create complex queries read about Django filter queries and Tastypie. Also a have a look at the Api Guide for a reference on how it all translates to this project in particular. The API is changing at a fast pace and the examples might be behind the current version.
